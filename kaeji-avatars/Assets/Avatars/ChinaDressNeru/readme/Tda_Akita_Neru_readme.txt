■説明
Tda様が作ったモデルを山本様がワンピースを着せたモデルを邪悪の化身バレッタ!!様がすっぴんにしたモデルを亞北ネルに改変しました。AGA様が改変した下腹も借りました。
このネル改変は只野まぐ様の絵から強い影響を受けました。

※自分は日本語を書くのが難しいので、変な言葉を使ったかもしれません。申し訳ありませんがご了承ください。


MMD杯9の後、いろいろ修正と変化しましたが、皆様は動画で見たネルさんが気に入ったと分かっています。
元姿が好きの皆様のために、モーフでカスタマイズできるバージョンも付けました。自由にお好みのネルさんを作ってください。
カスタマイズなしバージョンはシンプルで、最後の修正です。他の衣装を作れば、そのバージョンをベースとして使うつもりです。

PMXモーフの所為でモデルが正しく表示されない場合は、MikuMikuEffectのスブセット展開を使って、材質を非表示に設定することができます。（例：生足モーフしても、タイツはまだ反射像に表示）
シェーダ使用場合（例：AdultShader, MassLight, Figure）、PMD Editorでおまけの「shaded_face00_material.csv」（Cバージョンの場合は「shaded_face00_material_C.csv」）が「材質」タブで「CSVファイルから追加／更新」を使用することをおすすめします。
あるいは、手動でFace00材質のセルフ影がオンにして、Toon設定が「tex/toon_skin.bmp」に変化することができます。

※MikuMikuOnlineに使う場合は、「basic_run_toeIKfix.vmd」をMikuMikuOnlineの「motions」フォルダーにコピーするのが必要です。

■すっぴんアペミクワンピースからの変更点

頂点：
・顔がわずかに広げて丸くなった
・鼻が撮んで上げた
・口が上げて狭くなった
・目の形が僅かに広げてつり目にされ
・まつげが余計な頂点を消除
・まゆげの形と厚さが変化
・前髪ともみあげが外へ縮らす、左のもみあげが短くなった
・頭髪モデルとテクスチャが改造
・新しいアホ毛が作った
・ポニーテールの形が大きくなった改造
・背が短くされ、プロポーションもちょっと小さくなった
・前腕と足の形が変化
・シャツとタイツ追加
・ドレスの下に下腹モデルが追加 （突き抜け発生を避けるためにデフォルトはオフ）
・靴の形が変化、ストラップも追加
・下顎と首の頂点を結合
・鈴、帯と髪飾りが削除
・胸をちょっと大きくなった（普通にミクさんより大きいので）

エッジ・法線：
・首の所でアペンド衣装の残りエッジが消除
・指と足に自動三角測量された凹なエッジを修正 （つま先はまだ凹みエッジがある）
・ポニーテールともみあげに不規則な法線を修正
・顔が下から見る変なりんかく線を修正ために、ほお・鼻・口の外周にエッジ倍率を０に設定

UV・テクスチャ・材質：
・テクスチャが色を変更され、化粧を少し復元した
・いろんなUVとテクスチャはリマップされ、合体されました（ワンピースのノーマルマップはサポートされていなくなった）
・顕著なUVとテクスチャエラーが修正（例：手指と足指の爪、爪の下側、重なったUV）
・可視なテクスチャ分割線を修正（例：指、手首、足首、腕と肩）
・材質設定を変更（例：スフィア、透明度、エッジ色とサイズ等）
・改変を簡単にするため、いろんな材質が分離した

ウェイト・ボーン・物理：
・首、肩、足首のウェイトが調整
・ある部分のウェイトがSDEFに変化
・腰ボーンが再追加、足IKが調整、ポニーテールにIKを付けた
・準標準ボーン追加
・ジョイントと剛体が調整、 上半身・ひざ・足首の 剛体も追加
・胸１ジョイントの接続剛体Ａが上半身から上半身２に修正した

モーフ：
・Tda式アペミクからの眼鏡、吉、他のモーフ（例：はちゅ目）を再追加
・いろんな表情モーフエラーが修正 （例：変なりんかく線、表情モーフにあった顔の全頂点が僅かに動くエラー）
・色と形変化モーフをたくさん追加



■既知の不具合
・首、肩、肱と足首の形がときどき変になるかも
・自動三角測量の所為で、ある角から見ると形が非常に凹なエッジがまだある （とくにつま先）
・エッジループが足りないから、ひざの変形がギザギザの形状に見える
・たくさんの表情モーフを重なりすれば、変なエッジとりんかく線が出るかも
・足がモーション中にときどきドレスを突き抜け（MikuMikuMovingの場合には可能性がもっと高いらしい）
・モーション中にときどきドレスとポニーテールから暗いエッジがゆらゆら出現
・上半身ボーンを前方へ曲がれば、背中が非常に平な形になる
・不規則な法線が不均一な表面シェーディングを引き起こす可能性がある
・下着はまだ下の部分で少しのUVエラーがある
・モーションに応じて、頭と首の回転が非常に不自然に見えるかも
・透明の所為で、影の異常がある（例：タイツの透明）
・ポニーテールのウェイトがちょっと変かも（とくに端のところ）
・モーション中にときどきポニーテールを掴んで抱き
・「ヒールなし頂点」モーフを使ったら、足先EXボーンの回転が変になる
・MikuMikuOnlineのモデルは旧い
・あるポーズでポニーテールの初め部分が変に曲がってしまう
